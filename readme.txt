WEB SITE: LibTomMath is a free open source portable number theoretic multiple-precision integer library written entirely in C.
          Visit the LibTomMath web site for the latest news and downloads:
          http://www.libtom.org
 
GIT: To download the very latest source off the GIT server do this:
        git clone https://github.com/libtom/libtommath.git


Using TortoiseGIT to check out the source code from git link to this folder by the default settings.

The folder ./win32/ contain the Visual studio 2010 project.
